GMOクリック証券とのhttps通信をハックする その1

GMOクリック証券のウェブサイトはFlashを多用しているので、通信を覗くのが難しい。

そこで、GMOのAndroidアプリの通信をハックしAPIを作成する。

例えば、外オプroidはhttps通信でjsonのデータをやりとりしているのでその内容を覗いてみる。

https通信を覗くにはパケットキャプチャソフトのサーバー署名をAndroidにインストールし、 中間者攻撃でやりとりを覗く。

詳しい方法はここを参照 : Burp Suiteを使ってスマホアプリの通信を調査する
